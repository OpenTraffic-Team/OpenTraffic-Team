## OpenTraffic-Vision

The vision repository contains the current public perception-side content of OpenTraffic, including:

1. `.so` delivery bundles
2. result examples
3. validation JSON files
4. release indexes
5. future perception skills

## OpenTraffic-Control

The control repository is reserved for future control-side contribution, including:

1. control algorithms
2. interfaces and deployment structure
3. experiments and evaluation records
4. future control skills
