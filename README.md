# 🚀 OpenTraffic Ecosystem

OpenTraffic 当前主要围绕两个方向组织公开项目：

- **OpenTraffic-Perception**（交通感知）
- **OpenTraffic-Control**（交通控制）

---

# 🚦 OpenTraffic-Perception

交通感知侧基础设施，包含感知模型、工具链与智能体系统。

## Repositories

| Repository | Description |
|---|---|
| [OpenTraffic-Vision](https://github.com/OpenTraffic-Team/OpenTraffic-Vision) | 感知侧公开资源、发布内容与示例 |
| [OpenTraffic-Perception-Engine](https://github.com/OpenTraffic-Team/opentraffic-perception-engine) | 感知模型层 |
| [OpenTraffic-Perception-Skills](https://github.com/OpenTraffic-Team/opentraffic-perception-skills) | 感知工具层 |
| [OpenTraffic-Perception-Agent](https://github.com/OpenTraffic-Team/opentraffic-perception-agent) | 感知智能体层 |

## Current Contents

- 感知模型推理
- 多目标检测与跟踪
- 边缘设备部署
- `.so` 发布与运行环境
- 验证 JSON 与结果样例
- 感知能力封装
- 智能感知 Agent

# 🧠 OpenTraffic-Control

交通控制侧基础设施，聚焦强化学习交通信号控制与控制系统工程。

## Repositories

| Repository | Description |
|---|---|
| [OpenTraffic-TSC-Agent](https://github.com/OpenTraffic-Team/opentraffic-tsc-agent) | 强化学习交通信号控制智能体 |
| [OpenTraffic-TSC-Skills](https://github.com/OpenTraffic-Team/opentraffic-tsc-skills) | 控制技能与策略模块 |
| [OpenTraffic-TSC-Engine](https://github.com/OpenTraffic-Team/opentraffic-tsc-engine) | 控制仿真与运行引擎 |
| [OpenTraffic-Ops](https://github.com/OpenTraffic-Team/opentraffic-ops) | 部署、运维与基础设施 |

## Current Contents

- Traffic Signal Control (TSC)
- Reinforcement Learning
- 多阶段信号控制
- 仿真与训练框架
- 控制技能系统
- 实验与评估体系
- 边缘控制部署
